# 考试刷题应用

一个手机优先的刷题网页应用，支持本地记录和多端云同步。

## 功能

- 首页入口：知识点、刷题、收藏、错题本、保存/同步
- 刷题分类：按题型、按业务分类
- 题型支持：单选、多选
- 自动记录每个分类的刷题位置
- 每轮刷完后再次刷题会重新随机题目顺序，并清空上一轮答题结果
- 支持收藏题目、错题本，错题答对 2 次自动移除
- 支持保存码导出/导入
- 支持同步码云同步：手机和电脑填写同一个同步码，即可同步收藏、错题本和刷题记录

## 本地启动

推荐使用带同步 API 的启动方式：

```bash
python server.py
```

访问：

```text
http://127.0.0.1:8080/index.html
```

手机和电脑在同一 Wi-Fi 下，访问：

```text
http://电脑局域网IP:8080/index.html
```

仅静态预览也可以运行：

```bash
python -m http.server 8080 --bind 0.0.0.0
```

但静态预览没有云同步 API，只能使用本机记录和保存码导入导出。

## 文件说明

- `index.html`：刷题应用主文件，包含界面、逻辑和题库数据
- `server.py`：静态页面服务和 SQLite 云同步 API
- `data/sync.sqlite3`：运行后自动生成的同步数据库，不提交到 Git
- `import_questions_to_app.py`：从整理后的 TXT 题库导入应用的脚本
- `其他题_未导入应用.txt`：未导入应用的其他/案例类题目
- `题库导入报告.json`：题库导入统计

## 云同步说明

打开应用后进入“保存/同步”：

1. 输入同步码，例如 `qingbei001`
2. 点击“启用同步”
3. 手机和电脑使用同一个同步码
4. 收藏、错题本、刷题记录会自动保存到服务器并在多端合并

同步码只用于区分数据，不是安全账号密码。请不要使用太容易被别人猜到的同步码。
