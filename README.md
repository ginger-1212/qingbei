# 考试刷题应用

一个手机优先的本地刷题网页应用，打开 `index.html` 即可使用。

## 功能

- 首页入口：知识点、刷题、收藏
- 刷题分类：按题型、按业务分类
- 题型支持：单选、多选
- 自动记录每个分类的刷题位置
- 每轮刷完后再次刷题会重新随机题目顺序
- 支持收藏题目和收藏复习
- 题库已内置到 `index.html`

## 本地访问

直接打开：

```text
index.html
```

手机访问时，可在本机目录启动静态服务：

```bash
python -m http.server 8080 --bind 0.0.0.0
```

然后在同一 Wi-Fi 下用手机访问：

```text
http://电脑局域网IP:8080/index.html
```

## 文件说明

- `index.html`：刷题应用主文件，包含界面、逻辑和题库数据
- `import_questions_to_app.py`：从整理后的 TXT 题库导入应用的脚本
- `其他题_未导入应用.txt`：未导入应用的其他/案例类题目
- `题库导入报告.json`：题库导入统计
